# VerseMate 

This project is built with **React, TypeScript, and Vite**. 
It includes **ESLint** for code quality and **Hot Module Replacement (HMR)** for a smooth development experience.  

## Tech Stack  
- **React + TypeScript** for UI development  
- **Vite** for fast builds and HMR  
- **ESLint** with recommended rules for code quality
